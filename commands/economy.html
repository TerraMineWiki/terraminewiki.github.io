Commands : /pay, /balance and /rewards

/pay

The Pay command opens an interface to select a player and an amount to give to the designated player.

Usage

/pay

/balance

The Balance command is used to check the amount of money you have in your account.

Usage

/balance

/rewards

The Rewards command is used to check the daily rewards you can claim.

Usage

/rewards
