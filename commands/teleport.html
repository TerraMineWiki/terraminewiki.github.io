Commands : Tpr, Rtp, Spawn and other..

/tpr

The /tpr command is used to teleport to others players, with their permission.

Usage

/tpr <player>

/rtp

The /rtp command is used to teleport to a randon position.

Usage

/rtp

/spawn

The /spawn command is used to teleport to the spawn location.

Usage

/spawn

/tphr

The /tphr is the reverse version of the /tpr command : it asks a player to tp to the asking player.

Usage

/tphr <player>

/tpr_mute

The /tpr_mute command is used to automatically hide teleport requests from a specific player.

Usage

/tpr_mute <player>

/tpr_unmute

The /tpr_unmute command is used to show teleport requests from a specific player that you previousely muted.

Usage

/tpr_unmute <player>
